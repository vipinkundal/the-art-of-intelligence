Roadmap snapshot

Follow The Sequence, Then Open The Library

The full lesson list lives in the lesson library. Start with the sequence when you want a practical order for building skill.

01 Math and classical ML Build regression, trees, boosting, PCA, k-means, and SVMs with NumPy. 02 Deep-learning basics Autodiff, MLPs, CNNs, RNNs, optimizers, normalization, and training loops. 03 Transformers from scratch Tokenizer, attention, transformer block, decoder model, sampling, and KV cache. 04 Retrieval system BM25, embeddings, hybrid search, reranking, citations, and faithfulness checks. 05 Post-training Compare prompting, SFT, LoRA, preference optimization, and verifier workflows. 06 Agent engineering Typed tools, explicit state, approval gates, tracing, and reproducible evaluation. 07 AI systems Profiling, memory, mixed precision, batching, quantization, serving, and monitoring. 08 Specialization Choose one area, reproduce papers, and attempt one original extension.

Learning path

From Vocabulary To Skill

The roadmap is intentionally practical: learn the words, build the algorithms, measure the systems, then specialize.

  1. 1

    Learn the language

    Start with core terms so articles, courses, and documentation become easier to read.

  2. 2

    Build implementation fluency

    Practice Python, tensor programming, systems tools, numerical precision, testing, and profiling.

  3. 3

    Ship working evidence

    Finish with retrieval, agents, serving, evaluation, paper reproductions, and portfolio artifacts.
