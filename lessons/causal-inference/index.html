21. Causal Inference

Causal inference studies how to reason about cause and effect, separating predictive correlation from interventions, counterfactuals, and decisions that change outcomes.

Causal reasoning workflow

Model assumptions Represent variables, arrows, hidden causes, and the question the data alone cannot settle.
Identify effects Use adjustment, instruments, discontinuities, panels, or experiments to isolate causal signal.
Estimate and decide Estimate treatment effects, quantify uncertainty, and check whether the answer supports action.

Predictive correlation is not causation

A model can predict an outcome from variables that are only associated with it. Causal inference asks what would happen if an intervention changed one variable while the rest of the system followed the causal structure.

Learn

  • Directed acyclic graphs: Represent causal assumptions with arrows and no directed cycles.
  • Structural causal models: Describe each variable as a function of its direct causes plus noise.
  • Confounding: Occurs when a common cause influences both the treatment and the outcome.
  • Colliders: Variables caused by two other variables; conditioning on them can create misleading associations.
  • Mediators: Variables that transmit part of a treatment effect from cause to outcome.
  • D-separation: Reads conditional independence from graph structure by checking whether paths are blocked.
  • Back-door criterion: Identifies adjustment sets that block non-causal paths from treatment to outcome.
  • Front-door criterion: Identifies effects through a mediator when direct confounding blocks back-door adjustment.
  • Interventions: Replace the usual mechanism for a variable with an externally set value.
  • Do-calculus: Uses rules for transforming interventional expressions into estimable observational quantities.
  • Counterfactuals: Ask what would have happened to the same unit under a different treatment or action.
  • Potential outcomes: Represent each unit's possible outcomes under different treatment states.
  • Average treatment effects: Measure the expected difference between treated and untreated potential outcomes.
  • Propensity scores: Estimate the probability of receiving treatment given observed covariates.
  • Inverse-probability weighting: Reweights units so treatment groups resemble a balanced pseudo-population.
  • Doubly robust estimation: Combines outcome modeling and propensity weighting so one correct model can still identify the effect.
  • Instrumental variables: Use a source of treatment variation that affects the outcome only through treatment.
  • Regression discontinuity: Estimates causal effects near a threshold that assigns treatment.
  • Difference-in-differences: Compares outcome changes over time between treated and comparison groups.
  • Causal discovery: Attempts to infer causal graph structure from data plus assumptions.

Interventions and effects

Average treatment effect ATE = E[Y(1) - Y(0)]
Back-door adjustment P(Y | do(X = x)) = sum_z P(Y | X = x, Z = z) P(Z = z)

Identification strategies

  • Randomized experiments: Break confounding by assigning treatment independently of potential outcomes.
  • Covariate adjustment: Controls for observed confounders that block back-door paths.
  • Matching: Compares treated and untreated units with similar covariates or propensity scores.
  • Weighting: Creates a balanced weighted population for treatment-effect estimation.
  • Instrumental-variable estimation: Uses exogenous treatment variation when direct adjustment is not enough.
  • Regression discontinuity designs: Focus on units close to an assignment cutoff.
  • Difference-in-differences designs: Use pre/post changes to remove stable baseline differences.
  • Sensitivity analysis: Tests how strong hidden confounding would need to be to change the conclusion.

Causal discovery

  • PC algorithm: Uses conditional-independence tests to recover a graph equivalence class under assumptions such as causal sufficiency.
  • GES: Searches over graph structures with a score-based greedy procedure.
  • NOTEARS: Casts DAG learning as continuous optimization with an acyclicity constraint.
  • Assumption checks: Causal discovery depends heavily on faithfulness, no hidden confounding, temporal order, and measurement quality.

Why it matters for AI

Causal reasoning is especially important for medicine, economics, policy, scientific discovery, and decision-making systems. It helps AI teams decide whether a model's recommendation would actually improve an outcome rather than merely predict who already has a high or low risk.

Learning target

You should be able to distinguish prediction from causal effect estimation, read simple DAGs, identify confounders, colliders, mediators, back-door and front-door conditions, explain interventions and counterfactuals, estimate average treatment effects with weighting or doubly robust methods, and recognize when IV, regression discontinuity, difference-in-differences, or causal discovery methods are appropriate.
