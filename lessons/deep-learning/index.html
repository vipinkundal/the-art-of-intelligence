6. Deep Learning Foundations

Deep learning foundations cover neural network computation, convolutional architectures, and sequence models that underpin modern vision, speech, language, and multimodal systems.

6.1 Neural networks

Study:

  • Perceptrons
  • Multilayer perceptrons
  • Computational graphs
  • Backpropagation
  • Activation functions
  • Weight initialization
  • Normalization
  • Residual connections
  • Regularization
  • Universal approximation
  • Representation learning

Important activations

ReLU ReLU(x) = max(0, x)
GELU GELU(x) = x Phi(x)
SiLU SiLU(x) = x sigma(x)

Layer normalization

LN(x) = gamma * ((x - mu) / sqrt(sigma^2 + epsilon)) + beta

Learn

  • Batch normalization
  • Layer normalization
  • RMSNorm
  • Dropout
  • Weight decay
  • Early stopping
  • Data augmentation
  • Gradient clipping
  • Label smoothing
  • Knowledge distillation

  • Translation equivariance
  • Depthwise and separable convolutions
  • Residual networks

Two-dimensional convolution

Y[i, j] = sum_m sum_n X[i + m, j + n] K[m, n]

Important architectures and ideas

  • LeNet
  • AlexNet
  • VGG
  • Inception
  • ResNet
  • DenseNet
  • EfficientNet
  • MobileNet
  • U-Net

6.3 Recurrent and sequence models

Study:

  • Recurrent neural networks
  • Backpropagation through time
  • Vanishing and exploding gradients
  • LSTM
  • GRU
  • Bidirectional RNNs
  • Sequence-to-sequence models
  • Encoder-decoder models
  • Attention
  • Connectionist temporal classification

Basic recurrence

h_t = phi(W_h h_{t-1} + W_x x_t + b)

RNNs remain important conceptually even though transformers and state-space models dominate many modern sequence applications.

Learning target

You should be able to explain feed-forward networks, activation and normalization choices, convolutional architectures, and recurrent sequence models, then connect those ideas to modern deep learning systems.
