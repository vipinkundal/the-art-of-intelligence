Deep Learning

Neural Networks

A neural network is a model made of layers that transform input data into useful predictions or generated outputs.

The simple idea

Each layer changes the data a little. Early layers may detect simple patterns, while later layers combine those patterns into more useful meaning.

Input layer Receives numeric features, pixels, tokens, audio frames, or embeddings.
Hidden layers Apply weights and activations to build progressively richer representations.
Output layer Produces class scores, a value estimate, generated tokens, or another task-specific result.

You should understand

  • Inputs and outputs: Inputs are encoded as numbers; outputs are the task result, such as probabilities, values, embeddings, or generated tokens.
  • Layers, weights, and activations: Weights control each transformation, while activation functions add non-linearity so the network can model complex patterns.
  • Depth: More layers let the network combine simple signals into higher-level features, but also make training stability and data quality more important.

Example

In image recognition, one layer might detect edges, another might detect shapes, and a later layer might help identify the object.

Why it matters for AI

Neural networks are flexible enough to work with images, audio, text, code, and many other forms of data, which makes them the core building block of modern deep learning.

Learning target

You should be able to explain a neural network as layered transformations from input to output and identify what weights and activations do.
