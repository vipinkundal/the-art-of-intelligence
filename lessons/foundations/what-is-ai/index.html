Foundations

What Is AI?

Artificial intelligence is software designed to do tasks that normally require human judgment, such as recognizing patterns, understanding language, making predictions, or generating content.

The simple idea

AI systems take input, process it using rules or learned patterns, and produce an output. The output might be a label, a recommendation, an answer, a generated image, or the next action in a workflow.

Think of it like this

  • A normal app follows instructions written directly by a programmer.
  • An AI app still has code, but part of its behavior comes from patterns learned from examples.
  • The useful question is not whether it feels intelligent, but what task it can perform and how reliably it performs it.

Example

A spam filter looks at email text, sender details, and past examples. It predicts whether a new message is safe or spam.

Why it matters for AI

This definition separates AI from ordinary software and gives every later topic a clear reference point: inputs, learned patterns, outputs, and reliability.

Learning target

You should be able to explain what makes a system AI, name the input and output of a simple AI system, and describe one way to judge whether it works.
