7. Transformers And Foundation Models

The transformer remains the central architecture for language and many multimodal systems. The Wikipedia transformer article is useful for an introductory structural overview, while original papers and current courses should be used for implementation details.

7.1 Transformer fundamentals

Learn:

  • Token embeddings
  • Positional encoding
  • Sinusoidal positions
  • Rotary positional embeddings
  • ALiBi
  • Queries, keys and values
  • Self-attention
  • Cross-attention
  • Multi-head attention
  • Masked causal attention
  • Feed-forward networks
  • Residual streams
  • Pre-norm and post-norm
  • Encoder-only models
  • Decoder-only models
  • Encoder-decoder models

Scaled dot-product attention

Attention(Q, K, V) = softmax((QK^T) / sqrt(d_k)) V

Multi-head attention

MHA(Q, K, V) = Concat(head_1, ..., head_h) W^O

Autoregressive factorization

p(x_1:T) = product_{t=1..T} p(x_t | x_{<t})

Perplexity

PPL = exp(-(1 / T) * sum_{t=1..T} log p(x_t | x_{<t}))

Empirical scaling law

L(N, D) ~= L_infinity + aN^{-alpha} + bD^{-beta}

N represents model size and D represents training data. These are empirical relationships, not universal physical laws.

Reward-model loss

L_RM = -log sigma(r_theta(x, y^+) - r_theta(x, y^-))

DPO optimizes preference data without fitting a separate reward model and then running a conventional RL loop.

RLHF has its own Wikipedia overview, but use the original papers and current post-training literature for implementation details.

7.6 Parameter-efficient adaptation

Learn:

  • Adapters
  • LoRA
  • QLoRA
  • Prefix tuning
  • Prompt tuning
  • P-tuning
  • Low-rank and sparse updates
  • Partial fine-tuning
  • Quantization-aware adaptation

LoRA

W' = W + (alpha / r) BA

BA is a trainable low-rank update and the original W can remain frozen.

  • Grammar-constrained decoding
  • Speculative decoding

  • Best-of-N
  • Reranking
  • Self-consistency

Temperature

p_i = exp(z_i / T) / sum_j exp(z_j / T)

Lower T sharpens the distribution; higher T increases diversity.

Learning target

You should be able to explain transformer blocks, tokenization choices, attention variants, pretraining objectives, post-training methods, parameter-efficient adaptation, and decoding strategies used by modern foundation models.
