Generative AI

Transformers

A transformer is a neural network design that helps models understand relationships across a sequence, such as words in a sentence or tokens in code.

The simple idea

Transformers use attention to decide which parts of the input are most relevant to each other. This helps the model connect context across long text.

Tokens Text, code, image patches, or other inputs are split into units the model can process.
Attention Each token scores which other tokens matter for interpreting or generating the next representation.
Contextual output The model emits representations or next-token probabilities shaped by the surrounding context.

You should understand

  • Tokens and sequences: Models process ordered token streams, so tokenization and context length shape what the model can represent.
  • Attention: Attention lets each token weigh other tokens, which helps the model resolve references, dependencies, and long-range context.
  • Foundation-model reach: The same architecture can process language, code, retrieval context, images, and audio when those inputs are represented as tokens or embeddings.

Example

In the sentence "The robot picked up the cup because it was empty," attention helps the model reason about what "it" refers to.

Why it matters for AI

Modern language models rely on transformers because they can process context, generate coherent text, write code, summarize documents, and power assistants.

Learning target

You should be able to explain attention as a mechanism for relating tokens and describe why transformers are central to generative AI.
