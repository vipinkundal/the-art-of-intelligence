Phase 5

5. Classical Machine Learning

Classical machine learning covers the supervised models, objectives, losses, calibration methods, and model families that still power many reliable production systems.

5.1 Supervised learning

Regression

Learn:

  • Ordinary least squares
  • Polynomial regression
  • Ridge regression
  • Lasso
  • Elastic net
  • Generalized linear models
  • Poisson regression
  • Quantile regression
  • Robust regression
  • Gaussian-process regression

Linear regression

y_hat = Xw + b

Mean squared error

L = (1 / n) * sum_i (y_i - y_hat_i)^2

Ridge objective

min_w ||Xw - y||_2^2 + lambda ||w||_2^2

Classification

Learn:

  • Logistic regression
  • Multinomial logistic regression
  • Perceptron
  • k-nearest neighbours
  • Naive Bayes
  • Linear discriminant analysis
  • Quadratic discriminant analysis
  • Support-vector machines
  • Kernel methods
  • Decision trees
  • Random forests
  • Extremely randomized trees
  • AdaBoost
  • Gradient boosting
  • XGBoost
  • LightGBM
  • CatBoost
  • Probability calibration

Sigmoid

sigma(z) = 1 / (1 + exp(-z))

Softmax

p_i = exp(z_i) / sum_j exp(z_j)

Binary cross-entropy

L = -(1 / n) * sum_i [y_i log p_i + (1 - y_i) log(1 - p_i)]

Multiclass cross-entropy

L = -sum_{c=1..C} y_c log p_c

SVM objective

min_{w,b} (1 / 2) ||w||^2 + C * sum_i max(0, 1 - y_i(w^T x_i + b))

5.2 Unsupervised learning

Clustering

Learn:

  • k-means and k-means++
  • k-medoids
  • Gaussian mixture models
  • Expectation-maximization
  • Agglomerative clustering
  • Divisive clustering
  • DBSCAN
  • HDBSCAN
  • OPTICS
  • Mean shift
  • Spectral clustering
  • Affinity propagation
  • BIRCH
  • Fuzzy c-means

k-means objective

min_{mu_k} sum_i ||x_i - mu_{c_i}||_2^2

Dimensionality reduction

Learn:

  • PCA
  • Kernel PCA
  • Independent component analysis
  • Non-negative matrix factorization
  • Factor analysis
  • Random projections
  • Multidimensional scaling
  • Isomap
  • Locally linear embedding
  • t-SNE
  • UMAP
  • Autoencoders

Use t-SNE and UMAP mainly for exploratory visualization; do not treat distances in their 2D plots as definitive evidence.

Other unsupervised methods

Include:

  • Topic modelling with latent Dirichlet allocation
  • Matrix factorization
  • Collaborative filtering
  • Apriori
  • FP-growth
  • ECLAT
  • Isolation forest
  • Local outlier factor
  • One-class SVM
  • Robust covariance estimation

5.3 Evaluation and generalization

Learn:

  • Train, validation and test splits
  • Cross-validation
  • Stratified sampling
  • Data leakage
  • Overfitting and underfitting
  • Regularization
  • Learning curves
  • Hyperparameter optimization
  • Grid search
  • Random search
  • Bayesian optimization
  • Successive halving
  • Population-based training

Classification metrics

Precision Precision = TP / (TP + FP)
Recall Recall = TP / (TP + FN)
F1 F1 = 2 * Precision * Recall / (Precision + Recall)

Also understand

  • Accuracy
  • Balanced accuracy
  • Specificity
  • ROC-AUC
  • PR-AUC
  • Log loss
  • Brier score
  • Expected calibration error
  • Matthews correlation coefficient
  • Confusion matrices
  • Cost-sensitive evaluation

Regression metrics

  • MAE
  • MSE
  • RMSE
  • R^2
  • MAPE and its limitations
  • Quantile loss

Ranking and retrieval metrics

  • Precision@k
  • Recall@k
  • Mean reciprocal rank
  • Mean average precision
  • NDCG

Learning target

You should be able to choose a regression or classification model, explain its objective, recognize the loss it optimizes, compare linear, kernel, tree, ensemble, and calibrated probabilistic classifiers, select unsupervised methods for clustering, dimensionality reduction, association mining, recommendation, and anomaly detection, and evaluate models without leaking data or over-trusting a single metric.
