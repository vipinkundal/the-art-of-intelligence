Machine Learning

Training A Model

Training is the process of showing a model examples so it can adjust itself and get better at a task.

The simple idea

A model starts with weak guesses. During training, it compares its guesses with the correct answers, measures the error, and updates itself to reduce that error.

Important terms

  • Example: one row, image, sentence, or record used for learning.
  • Label: the correct answer attached to an example.
  • Loss: a score that tells the model how wrong it was.

Example

To train a model that predicts house prices, you show it many houses with size, location, room count, and final sale price.

Why it matters for AI

Training is the step where data changes model behavior. Understanding it helps you reason about errors, overfitting, evaluation, and why better examples can improve predictions.

Learning target

You should be able to describe the training loop in plain language: make a guess, measure loss, update the model, and check whether performance improves.
