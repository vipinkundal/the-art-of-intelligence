1. Mathematical Foundations

1.2 Calculus And Matrix Calculus

Calculus explains how quantities change. Matrix calculus extends that idea to vectors, matrices, losses, gradients, Jacobians, and Hessians, which are central to training AI models.

The simple idea

Calculus gives AI a way to ask: if we change an input, parameter, or weight a little, how does the output or loss change? Optimization uses that answer to improve models.

Change Derivatives measure how one quantity responds when another quantity moves.
Sensitivity Gradients, Jacobians, and Hessians scale the idea to many inputs and outputs.
Training Backpropagation uses these sensitivities to update model weights.

Learn

  • Limits and continuity: Limits describe behavior near a point, and continuity means small input changes do not cause jumps.
  • Derivatives and partial derivatives: Derivatives measure change; partial derivatives isolate one variable while holding others fixed.
  • Product, quotient, and chain rules: These rules break complex derivatives into manageable pieces, especially through nested functions.
  • Gradients: A gradient collects partial derivatives and points in the direction of steepest local increase.
  • Directional derivatives: These measure how a function changes along one chosen direction instead of all directions.
  • Jacobians: A Jacobian stores first-order derivatives for vector-valued functions, tracking how each output changes with each input.
  • Hessians: A Hessian stores second-order derivatives and describes curvature, sharpness, and local optimization geometry.
  • Taylor approximation: Taylor formulas approximate a function near a point using value, slope, and sometimes curvature.
  • Multivariable integration: Integrates quantities over areas, volumes, or higher-dimensional spaces.
  • Change of variables: Re-expresses integrals or densities in new coordinates while correcting for volume distortion.
  • Ordinary differential equations: Describe systems whose state changes over continuous time according to deterministic rules.
  • Stochastic differential equations: Add random noise to continuous-time dynamics, useful for diffusion and uncertainty models.
  • Automatic differentiation: Computes exact derivatives of programs by applying calculus rules to the computation graph.
  • Forward-mode and reverse-mode autodiff: Forward mode is efficient for few inputs; reverse mode is efficient for many parameters and one loss.

Essential formulas

Chain rule dL/dx = (dL/dy)(dy/dx)
First-order Taylor approximation f(x + Delta x) ~= f(x) + grad f(x)^T Delta x
Second-order approximation f(x + Delta x) ~= f(x) + grad f(x)^T Delta x + 1/2 Delta x^T H_f(x) Delta x

Why it matters for AI

Training models means changing parameters to reduce a loss. Gradients tell us the direction of change, Jacobians describe vector-valued sensitivity, and Hessians describe curvature.

Backpropagation is mostly the repeated application of the chain rule over a computational graph.

Learning target

After this lesson, you should be able to explain gradients, partial derivatives, Jacobians, Hessians, Taylor approximations, and why reverse-mode automatic differentiation is useful for neural network training.
