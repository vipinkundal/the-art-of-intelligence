Phase 4

4. Probabilistic AI

Probabilistic AI gives models a language for uncertainty, hidden state, noisy evidence, and inference over structured variables.

4.1 Probabilistic graphical models

Study:

  • Bayesian networks
  • Markov random fields
  • Factor graphs
  • Conditional random fields
  • Conditional independence
  • D-separation
  • Exact inference
  • Approximate inference

Important algorithms

  • Variable elimination
  • Sum-product algorithm
  • Max-product algorithm
  • Belief propagation
  • Loopy belief propagation
  • Junction-tree algorithm
  • Expectation-maximization
  • Variational inference
  • Gibbs sampling
  • Metropolis-Hastings
  • Hamiltonian Monte Carlo
  • Sequential Monte Carlo

4.2 Sequential models and state estimation

Learn:

  • Markov chains
  • Hidden Markov models
  • Forward-backward algorithm
  • Viterbi decoding
  • Kalman filter
  • Extended Kalman filter
  • Unscented Kalman filter
  • Particle filter
  • Dynamic Bayesian networks

HMM joint distribution

p(z_1:T, x_1:T) = p(z_1) * product_{t=2..T} p(z_t | z_{t-1}) * product_{t=1..T} p(x_t | z_t)

Kalman prediction

State mean x_hat_{t|t-1} = A x_hat_{t-1|t-1} + B u_t
State covariance P_{t|t-1} = A P_{t-1|t-1} A^T + Q

Learning target

You should be able to represent uncertainty with graphical models, choose an inference algorithm, explain how sequence models track hidden state over time, and recognize when filtering or smoothing is the right tool.
