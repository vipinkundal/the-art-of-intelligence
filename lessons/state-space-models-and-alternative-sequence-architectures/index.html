13. State-Space Models And Alternative Sequence Architectures

State-space and convolutional sequence architectures are efficient alternatives and complements to attention for long-context modelling.

Sequence architecture map

State Compress past inputs into a hidden state that can be updated step by step.
Long context Use structure, convolutions, or selective updates to scale beyond quadratic attention.
Hybrid design Combine attention, recurrence, convolution, and gating where each is strongest.

Study

  • Linear dynamical systems: Model a hidden state that evolves through linear updates and emits observations over time.
  • State-space models: Track sequence history with a compact recurrent state instead of comparing every token to every other token.
  • Structured state spaces: Constrain the state update matrices so long sequences can be processed efficiently and stably.
  • S4: A structured state-space architecture designed to model very long dependencies with efficient convolutional computation.
  • Mamba: Uses selective state-space updates so the model can choose which information to keep or forget based on the input.
  • Selective state-space models: Make state updates input-dependent, improving expressiveness compared with fixed linear dynamics.
  • Gated convolutions: Use gates to control which convolutional features pass through a sequence model.
  • Hyena-style long convolutions: Replace attention with implicit long filters that can mix information over large contexts.
  • Retention mechanisms: Blend recurrent memory and attention-like weighting to keep useful historical information.
  • Transformer-SSM hybrids: Combine attention blocks with state-space or convolutional blocks for better efficiency-quality trade-offs.

Basic state-space form

h_t = A h_{t-1} + B x_t
y_t = C h_t + D x_t

These models are relevant to long-sequence processing and efficient alternatives to quadratic attention. They should be learned after transformers, not instead of transformers.

Learning target

You should be able to explain the recurrent state update, the output projection, why structured state spaces can scale to long sequences, and where SSMs, gated convolutions, retention, and transformer hybrids fit relative to attention.
